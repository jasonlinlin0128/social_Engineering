# 演練後說明公告文案（群組移轉版）

> 用途：演練發送後 **1 小時內** 在同一個 LINE 群發出，向同仁說明這是資安演練。
> 對應 2026 年「群組移轉」情境（非 2025 振興加倍金版）。發送前把 XX 換成實際數字。

---

## 主要版本（LINE 群直接貼）

```
📢【重要說明】剛才的「群組移轉」訊息是一次資安演練

各位同仁好，

今日稍早在群組收到的「教學討論群即將關閉，請點擊連結移轉至官方專屬群組」訊息，
是管理部依 113/12/20 資訊安全會議決議所執行的『防範惡意連結』社交工程演練，
並非真實通知，也沒有任何新群組。

🎯 演練目的
1. 檢視同仁對「群組異動／限時移轉」類訊息的警覺性
2. 提升對來路不明連結的辨識能力
3. 強化全體資安意識

📊 本次結果
• 共 XX 位同仁點擊了連結（點擊率約 XX%）
• 已點擊的同仁我們會做個別資安提醒，這不是處分，請不用擔心

💡 這則訊息的破綻（下次請留意）
1. 製造急迫感：「本週五解散」「現在順手點」逼你立刻行動
2. 假借單位名義：抬出管理部／經企室／EIP 讓訊息看起來很官方
3. 用真實活動包裝：提到你真的參加過的 AI-first 工作坊，降低戒心
4. 來源未查證：不是官方公告管道，只丟一個連結

🔒 收到類似訊息時
• 先暫停，別急著點
• 透過官方管道查證（問主管、IT，或當面／電話問講師本人）
• 覺得可疑就回報 IT 或管理部

感謝大家的配合，我們會在本週的資安教育訓練分享更多真實案例。

管理部 敬上
```

## 簡短版（若主管希望低調處理）

```
📢 說明一下：剛才群組那則「教學群即將關閉、請點連結移轉新群」的訊息，
是管理部進行的資安演練，不是真的，請大家放心，也沒有要解散群組。

重點提醒：以後遇到「限時、要你馬上點連結」的訊息，先停一下、透過官方管道查證再說。
有點到的同仁不用擔心，我們會另外做提醒。謝謝配合 🙏
```

---

## 發送注意

- 把 `XX` 換成後台統計的實際人數與點擊率（後台 `/admin.html` 有「名單比對」可直接看 已點擊／未點擊／點擊率）。
- 口吻以「教育、不究責」為主，避免讓點擊的同仁有壓力。
- 發送後保留後台資料，供資安教育訓練與成果報告使用。
