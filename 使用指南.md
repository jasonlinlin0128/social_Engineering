## 🔄 部署步驟

### 步驟 1：在 Render 重新部署

1. 前往：https://dashboard.render.com/

1. 前往 [LINE Developers Console](https://developers.line.biz/console/)
2. 找到你的 LIFF 應用（ID: `2008536456-eRYM8pMv`）
3. 點擊 **"Edit"**

6. 點擊 **"Update"** 儲存

## 📊 監控與分析

### 管理後台功能：

1. **即時監控**
   - 查看誰點擊了連結
   - 查看點擊時間
   - 查看點擊率統計

2. **資料匯出**
   - 點擊「📥 匯出 CSV」按鈕
   - 下載完整點擊記錄
   - 用於後續分析

3. **選擇性刪除**
   - 勾選要刪除的記錄
   - 點擊「🗑️ 刪除選中項目」
   - 可以清理測試資料

### 重要數據：

- **總點擊次數**：有多少人點了
- **不重複用戶**：有多少人（扣除重複點擊）
- **點擊率**：點擊人數 ÷ 群組人數
- **點擊時間分布**：什麼時候點擊最多

## 📞 技術支援

### 如果遇到問題：

**問題 1：連結預覽不正確**
- 解決：清除 LINE 快取，重新傳送連結
- 或：等待 10-15 分鐘讓 LINE 更新快取

**問題 2：點擊後顯示錯誤**
- 檢查：Render 服務是否正常運行
- 檢查：LIFF Endpoint URL 是否正確

**問題 3：管理後台沒有記錄**
- 檢查：是否在 LINE 中開啟（不是瀏覽器）
- 檢查：網路連線是否正常
- 檢查：後端 API 是否運作

### 檢查清單：

- [ ] Render 服務狀態為 "Live"
- [ ] LIFF Endpoint URL 設定為 `index.html`
- [ ] Scope 包含 `profile`
- [ ] 管理後台可以正常開啟
- [ ] 測試連結可以正常顯示預覽
- [ ] 點擊後會記錄到後台
