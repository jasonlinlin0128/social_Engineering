`https://liff.line.me/2008536456-eRYM8pMv` 這個網址：

## 方法 1：使用 reurl.cc（推薦）

### 步驟 1：前往 reurl.cc

網址：https://reurl.cc/

### 步驟 2：建立短網址

1. 在「請輸入網址」欄位貼上：
   ```
   https://liff.line.me/2008536456-eRYM8pMv
   ```

2. 點擊「自訂網址」

3. 輸入自訂名稱（建議使用）：
   ```
   benefit2024
   或
   employeebonus
   或
   10to50k
   ```

4. 點擊「縮短網址」

5. 得到短網址，例如：
   ```
   https://reurl.cc/benefit2024
   ```

### 步驟 3：測試短網址

1. 複製短網址
2. 在 LINE 中傳送給自己
3. 查看預覽是否正常顯示
4. 點擊確認會導向 LIFF 頁面

## 方法 2：使用 bit.ly（可自訂更完整）

### 步驟 1：註冊 bit.ly

1. 前往：https://bit.ly/
2. 使用 Google 帳號快速註冊（免費）

### 步驟 2：建立短網址

1. 登入後點擊「Create」
2. 貼上你的 LIFF 網址：
   ```
   https://liff.line.me/2008536456-eRYM8pMv
   ```

3. 點擊「Customize back-half」自訂網址

4. 輸入自訂名稱：
   ```
   company-benefit-2024
   或
   employee-bonus
   或
   cash10to50
   ```

5. 點擊「Create」

6. 得到短網址，例如：
   ```
   https://bit.ly/company-benefit-2024
   ```

### 進階功能：

bit.ly 還提供：
- 📊 點擊次數統計
- 📍 點擊來源分析
- 🕐 點擊時間分布

## 方法 3：使用 pse.is（PTT 短網址）

### 步驟：

1. 前往：https://pse.is/
2. 貼上 LIFF 網址
3. 點擊「縮短」
4. 會自動產生隨機網址，例如：`https://pse.is/6m9xyz`
5. **注意**：無法自訂網址

## ⚠️ 注意事項

### 1. 測試短網址

建立短網址後，**一定要先測試**：
- 在 LINE 中發給自己
- 確認預覽訊息正確
- 確認點擊後會導向 LIFF 頁面

### 2. 保留短網址

- 記錄你建立的短網址
- 避免遺失無法管理
- bit.ly 可以登入後台管理

### 3. 避免太明顯的名稱

❌ 不要用：
- `phishing-test`（太明顯）
- `fake-link`（太明顯）
- `social-engineering`（太明顯）

✅ 建議用：
- `benefit2024`（看起來正常）
- `employee-bonus`（看起來正常）
- `hr-update`（看起來正常）
